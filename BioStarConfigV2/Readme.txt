This tool is for v2 devices only. 

Note:
- the normal search feature searches only v2 devices 
- the advanced search feature cannot distinguish v1 and v2 devices
- if a v1 device is searched, it will appear as unknown and the configuration cannot be read. 
- if you upgraded a device with a 5 digit serial number, the device name will appear as unknown but the configuration can be read and written
- if the device is already connected to another server, the configuration cannot be written even if you click apply.
